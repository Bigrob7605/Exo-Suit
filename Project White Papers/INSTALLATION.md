## **V5.0 INSTALLATION METHODS**

### **Method 1: V5.0 Quick Start (Recommended)**

#### **Step 1: Clone Repository**
```bash
git clone <repository-url>
cd "Agent Exo-Suit"
```

#### **Step 2: Run V5.0 Auto-Setup**
```powershell
# Windows PowerShell - V5.0 Enhanced
.\AgentExoSuitV4.ps1 -FullSystem -V5Mode

# Or use the go-big script with V5.0 optimization
.\go-big.ps1 -V5Mode
```

#### **Step 3: Activate V5.0 Ultimate Overclock**
```powershell
# Activate V5.0 Ultimate Overclock System
.\ops\Ultimate-Overclock-Speed-Boost-V5.ps1 -Mode "Ultimate" -Benchmark

# Validate V5.0 performance
.\ops\Performance-Test-Suite-V4.ps1 -Mode "Extreme" -Benchmark
```

#### **Step 4: Verify V5.0 Installation**
```powershell
# Test all V5.0 components
.\Cleanup\Testing_Tools\test-runner.ps1 -V5Mode

# Validate V5.0 performance claims
.\ops\GPU-Monitor-V4.ps1 -Mode "V5Benchmark"
```

### **Method 2: V5.0 Manual Installation**

#### **Step 1: V5.0 Environment Setup**
```bash
# Create V5.0 optimized virtual environment
python -m venv gpu_rag_env_v5
source gpu_rag_env_v5/bin/activate  # Linux/macOS
# or
gpu_rag_env_v5\Scripts\activate     # Windows
```

#### **Step 2: Install V5.0 Core Dependencies**
```bash
# Install V5.0 enhanced requirements
pip install -r rag\requirements_hybrid_v4.txt

# For V5.0 GPU acceleration (CUDA 11.8+)
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

# For V5.0 GPU FAISS optimization
pip install faiss-gpu
```

#### **Step 3: Install V5.0 Additional Dependencies**
```bash
# V5.0 performance optimization packages
pip install psutil memory-profiler gputil

# V5.0 monitoring and optimization
pip install py-cpuinfo py3nvml
```
